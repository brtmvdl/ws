## Docker setup

Run with Docker

```sh
docker run -d -p 8000:80 tmvdl/ws
```

View on browser

http://localhost:8000/

## Manual setup

Install [Node.js](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com/)

Clone this repository

```sh
git clone https://github.com/tmvdl/ws.git
cd ws
```

Install the dependencies

```sh
yarn
```

Start the app

```sh
yarn start
```

View on browser

http://localhost:4200/
